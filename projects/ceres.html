Ceres

  • Category: IOS AR App
  • Technologies: Swift and Reality Composer
  • Video Demo: Demo
  • Github: Ceres_AR_App

Project Description

Ceres is an IOS Augmented Reality experirence, whereby a user can find out all the information about their ceres fruit juice. All the user needs to do is scan their fruit juice box and then user will then be taken through an AR experirence, to learn more about the product.
