Project Description

Dragonite is a cross-platform mobile application built in C# using Xamarin. The idea of the application was for the user to raise their very own dragon firstly from an egg where they need to keep it at the right tempreature, then once the egg has hatched the user will need to care for their dragon by feeding it as well as letting it sleep. The user can then further level up their dragon by either letting it go hunt or by training it in one of the three training options.
