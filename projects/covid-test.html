Project Description

The idea of the application is that a user would be able to upload and image of an x-ray and then be able to get a prediction and whether or not that x-ray could have Covid-19 or not. This application operates through the use of a deep learning model that has been trained on roughly 200 images of healthy lung X-Rays and then roughly 200 Covid-19 postivie X-rays. The model was trained using tensorflow and keras which are both free open source software libraries used for machine learning. There is alot of room for improvement in terms of the accuracy of the model. As at the moment the model is only about 90% accurate. And for medical purposes that would then need to be more in the regions of 98% to 100% accurate.
